Magnifying Glass Fisheye Warp Image Effect README

=== Intro ===

You can get a fullscreen Magnifying Glass Fisheye Warp camera image effect with this asset. Please goto AssetStore to wathcing the demo video.

=== How to use ===

1. Add the Image Effect Script to your Main Camera.

2. Play scnene, you will see the fullscreen Magnifying Glass Fisheye Warp image effect.

3. There are some custom variable that you can set as your like to get different effects.

=== Compatiability === 

We have test this on PC, MAC, IOS and Android. Each platform works well and get good performance.
